# Bullet Journal


## Overview
A bullet journal to keep track of goals. 


## Features
- Create goals
- View goals
- Update goals
- Delete goals 


## Technologies 
- Next 13
- Jest


## Running The Project

From the repo:<br/>
  - Clone this project locally<br/>
  - Install dependencies using `npm install`<br/>
  - `npm run dev` <br/>
